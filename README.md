Pár užitečných skriptů pro WordPress
htaccess.php - Přidá do .htaccess pravidla pro přístup k souboru wp-login.php pomocí jména a hesla.
htpasswd.php - Vytvoří nebo upraví soubor .htpasswd v případě že nemáme možnost použít příkaz htpasswd
vlozadmina.php - Vytvoří WordPress uživatele s právy administrátora.

Some useful scripts for WordPress
htaccess.php - Adds rules to .htaccess for accessing the wp-login.php file using a username and password.
htpasswd.php - Creates or modifies the .htpasswd file if we don't have the option to use the htpasswd command
vlozadmina.php - Creates a WordPress user with administrator rights.
